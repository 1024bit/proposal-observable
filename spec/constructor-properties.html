Observable.prototype

The initial value of `Observable.prototype` is the intrinsic object %ObservablePrototype%.

This property has the attributes { [[Writable]]: `false`, [[Enumerable]]: `false`, [[Configurable]]: `false` }.

Observable.from ( x )

When the `from` method is called, the following steps are taken:

1. Let _C_ be the `this` value. 1. If IsConstructor(C) is `false`, let _C_ be %Observable%. 1. Let _observableMethod_ be GetMethod(_x_, `@@observable`). 1. ReturnIfAbrupt(_observableMethod_). 1. If _observableMethod_ is not `undefined`, 1. Let _observable_ be Call(_observableMethod_, _x_, «‍»). 1. ReturnIfAbrupt(_observable_). 1. If Type(_observable_) is not Object, throw a `TypeError` exception. 1. Let _constructor_ be Get(_observable_, `"constructor"`). 1. ReturnIfAbrupt(_constructor_). 1. If SameValue(_constructor_, _C_) is `true`, return _observable_. 1. Let _subscriber_ be a new built-in function object as defined in Observable.from Delegating Functions. 1. Set the value of the [[Observable]] internal slot of _subscriber_ to _observable_. 1. Else, 1. Let _subscriber_ be a new built-in function object as defined in Observable.from Iteration Functions. 1. Set the value of the [[Items]] internal slot of _subscriber_ to _x_. 1. Return Construct(_C_, «‍_subscriber_»).

Observable.from Delegating Functions

An Observable.from delegating function is an anonymous bulit-in function that has an [[Observable]] internal slot.

When an Observable.from delegating function is called with argument _observer_, the following steps are taken:

1. Let _observable_ be the value of the [[Observable]] internal slot of _F_. 1. Return Invoke(_observable_, `"subscribe"`, «‍_observer_»).

The `length` property of an Observable.from delegating function is `1`.

Observable.from Iteration Functions

An Observable.from iteration function is an anonymous built-in function that has an [[Items]] internal slot.

When an Observable.from iteration function is called with argument _observer_, the following steps are taken:

1. Let _items_ be the value of the [[Items]] internal slot of _F_. 1. Let _cleanup_ be a new built-in function object as defined in Subscriber Closed Functions. 1. Set the value of the [[SubscriptionClosed]] internal slot of _cleanup_ to `false`. 1. Perform EnqueueJob(`"PromiseJobs"`, ObservableFromJob, «‍_observer_, _items_, _cleanup_»). 1. Return _cleanup_.

The `length` property of an Observable.from iteration function is `1`.

ObservableFromJob ( observer, items, cleanupFunction )

1. If the value of the [[SubscriptionClosed]] internal slot of _cleanupFunction_ is `true`, NextJob `undefined`. 1. Let _iterator_ be GetIterator(_observer_). 1. If _iterator_ is an abrupt completion, NextJob Invoke(_observer_, `"error"`, «‍_iterator_.[[value]]»). 1. Let _iterator_ be _iterator.[[value]]. 1. Repeat 1. Let _next_ be IteratorStep(_iterator_). 1. If _next_ is an abrupt completion, NextJob Invoke(_observer_, `"error"`, «‍_next_.[[value]]»). 1. Let _next_ be _next_.[[value]]. 1. If _next_ is `false`, NextJob Invoke(_observer_, `"complete"`, «‍»). 1. Let _nextValue_ be IteratorValue(_next_). 1. If _nextValue_ is an abrupt completion, NextJob Invoke(_observer_, `"error"`, «‍_nextValue_.[[value]]»). 1. Let _nextValue_ be _nextValue_.[[value]]. 1. Let _result_ be Invoke(_observer_, `"next"`, «‍_nextValue_»). 1. If _result_ in an abrupt completion, NextJob IteratorClose(_iterator_, _result_). 1. If the value of the [[SubscriptionClosed]] internal slot of _cleanupFunction_ is `true`, NextJob `undefined`.

Observable.of ( ...items )

1. Let _C_ be the `this` value. 1. If IsConstructor(C) is `false`, let _C_ be %Observable%. 1. Let _subscriber_ be a new built-in function object as defined in Observable.of Subscriber Functions. 1. Set the value of the [[Items]] internal slot of _subscriber_ to _items_. 1. Return Construct(_C_, «‍_subscriber_»).

Observable.of Subscriber Functions

An Observable.of subscriber function is an anonymous built-in function that has a [[Items]] internal slot.

When an Observable.of subscriber function is called with argument _observer_, the following steps are taken:

1. Let _items_ be the value of the [[Items]] internal slot of _F_. 1. Let _cleanup_ be a new built-in function object as defined in Subscriber Closed Functions. 1. Set the value of the [[SubscriptionClosed]] internal slot of _cleanup_ to `false`. 1. Perform EnqueueJob(`"PromiseJobs"`, ObservableOfJob, «‍_observer_, _items_, _cleanup_»). 1. Return _cleanup_.

The `length` property of an Observable.of subscriber function is `1`.

Subscriber Closed Functions

A subscriber closed function is an anonymous built-in function that has a [[SubscriptionClosed]] internal slot.

When a subscriber closed function is called the following steps are taken:

1. Set the value of the [[SubscriptionClosed]] internal slot of _F_ to `true`. 1. Return `undefined`.

The `length` property of a subscriber closed function is `0`.

ObservableOfJob ( observer, items, cleanupFunction )

1. If the value of the [[SubscriptionClosed]] internal slot of _cleanupFunction_ is `true`, NextJob `undefined`. 1. Let _len_ be the number of elements in _items_. 1. Let _k_ be 0. 1. Repeat, while _k_ < _len_, 1. Let _kValue_ be _items[k]_. 1. Let _nextResult_ be Invoke(_observer_, `"next"`, «‍_kValue_»). 1. If _nextResult_ is an abrupt completion, NextJob _nextResult_. 1. If the value of the [[SubscriptionClosed]] internal slot of _cleanupFunction_ is `true`, NextJob `undefined`. 1. Increase k by 1. 1. Let _result_ be Invoke(_observer_, `"complete"`, «‍»). 1. NextJob Completion(_result_).
